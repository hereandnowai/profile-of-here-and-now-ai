üîó Connect to Gemini AI

Time to connect to Google's Gemini AI! We need to tell our code where to find the AI and how to access it. üîê

  • base_url: The web address where Gemini AI service lives

üîë Getting Your API Key:

3. Add a secret named GEMINI_API_KEY with your key
4. Enable notebook access

üß† Build the Chat Logic

This function is the brain of your chatbot! üß† It handles every message the user sends.

üìù How it works:
  1. Start with personality: Add the system prompt first
  2. Include history: Remember previous messages in the conversation
  3. Add user message: Append the new message from the user
  4. Call the AI: Send everything to Gemini for processing
  5. Return response: Extract and return the AI's answer
  6. Update history: Keep track for next time
üí° Why history matters:
By including previous messages, the AI can understand context ‚Äì like "Tell me more about that" or "What did you just say?"
