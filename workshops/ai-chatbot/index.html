STEP 3 of 6

Connect to Gemini AI üîó

Time to connect to Google's Gemini AI!

We need to tell our code where to find the AI (base_url) and how to access it (API key). It's like entering an address and password to log in! üîê

base_url = "https://generativelanguage.googleapis.com/v1beta/openai/"
api_key = userdata.get("GEMINI_API_KEY")
client = OpenAI(base_url=base_url, api_key=api_key)
model = "gemini-2.5-flash-lite"

üß© What each line does:

  • base_url: The web address where Gemini AI lives
  • api_key: Your secret key stored in Colab (keeps it safe!)
  • client: Creates a connection to the AI service
  • model: Chooses which version of Gemini to use (we're using the fast "flash-lite" version!)

üîë Getting Your API Key

Visit ai.google.dev to get a free Gemini API key, then store it in Colab's "Secrets" section under the name GEMINI_API_KEY.

STEP 5 of 6

Build Conversation Logic üß†

Now we create the brain of our chatbot!

This function handles the conversation flow: taking user messages, sending them to Gemini, and returning AI responses. üí¨

def get_response(message, history):
messages = [{"role":"system", "content":ai_teacher}]
messages.extend(history)
messages.append({"role":"user", "content":message})
response = client.chat.completions.create(model=model, messages=messages)
ai_response = response.choices[0].message.content
return ai_response

üîÑ How it works:

  • Line 1: Define a function that takes user message & conversation history
  • Line 2: Start with the system prompt (AI personality)
  • Line 3: Add previous conversation history (so AI remembers context)
  • Line 4: Add the new user message
  • Line 5: Send everything to Gemini and get a response
  • Line 6: Extract the AI's text response
  • Line 7: Return the response to display in the chat

üéØ Key Concept

The conversation history lets your AI remember what was said earlier ‚Äì making it feel more natural and human-like! üßë‚Äçüíª

STEP 6 of 6

Create Beautiful UI üé®

Final step ‚Äì bring it all together with a stunning interface!

Gradio makes it super easy to create a professional chat UI with just a few lines. ‚ú®

PAGE_TITLE = "Caramel AI from HERE AND NOW AI !"
LOGO_URL = "https://raw.githubusercontent.com/hereandnowai/images/..."
ASSISTANT_AVATAR_URL = "https://raw.githubusercontent.com/hereandnowai/images/..."
 
description_md = f"""<img src='{LOGO_URL}' width='500' style='display: block; margin: auto;'>
<br>Your friendly AI teacher for learning the basics of AI!"""
 
gr.ChatInterface(
fn=get_response,
chatbot=gr.Chatbot(type='messages', avatar_images=[None, ASSISTANT_AVATAR_URL]),
title=PAGE_TITLE,
description=description_md,
examples=["What is AI?", "What does GPT stand for?", "Explain Neural Network!"],
type='messages'
).queue().launch(show_error=True, ssr_mode=False)

üé® UI Customization:

  • title: The main heading of your app
  • description: Text/HTML shown at the top (we add the HERE AND NOW AI logo!)
  • avatar_images: Custom profile pictures for user & AI
  • examples: Suggested questions users can click to try
  • .launch(): Opens the chat interface in a new window! üöÄ
